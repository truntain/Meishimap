CHANGELOG - Plan A: MapLibre + OpenFreeMap cho màn Search

File này ghi lại toàn bộ thay đổi khi triển khai bản đồ cho màn /search. Mục tiêu là để nhóm dễ đối chiếu: file nào được thêm/sửa, tên biến/API nào mới, dữ liệu đi qua những bước nào, và phần nào chưa đụng tới.

Phạm vi: chỉ làm màn Search. Không làm Google Places, không routing/chỉ đường, không import dữ liệu thật từ bên ngoài, không sửa các màn booking/review/detail nếu không cần.

1. Trạng thái trước khi sửa

  • Frontend /search đang dùng dữ liệu mock trong SearchClient.tsx.
  • Khu vực bản đồ bên phải là placeholder tự dựng, chưa có bản đồ thật.
  • Database đã có bảng restaurants với các cột tọa độ latitude, longitude.
  • Backend chưa có module/API cho restaurants.

2. Kế hoạch thay đổi

Hạng mục Dự kiến thay đổi Lý do
Backend Thêm entity/service/controller/module cho restaurants và API GET /restaurants. Lấy dữ liệu thật từ PostgreSQL thay cho mock frontend.
Frontend dependency Thêm maplibre-gl. Render bản đồ vector bằng MapLibre GL JS.
Frontend Search Đổi SearchClient.tsx sang fetch API backend, render MapLibre/OpenFreeMap, marker, popup, flyTo. Biến khu vực map placeholder thành bản đồ thật có tương tác.
